# Outputs — report structure

The full template lives in **`SKILL.md`** (Report format). Required **top-level sections** in order:

1. `# Trend Intelligence Report: [Company Name]` — metadata line (website, report date, industry).
2. `## Company overview` — narrative + bulleted **Core services identified**.
3. `## Top trending topics for your business` — **6–10** `###` subsections with Relevance, Trend direction, Why it matters, Opportunity.
4. `## Industry momentum summary` — paragraph + **Growth signals** + **Caution signals** bullets.
5. `## Keyword trends breakdown` — Markdown **table** (Topic, Direction, Momentum, Notes).
6. `## Strategic recommendations` — Quick wins, Medium-term, Watch list.
7. `## Trends to avoid / de-prioritize` — 2–3 items.
8. `## Sources & research basis` — Bullets with **`[title](https://...)`** per source.

**Direction labels:** Use **Rising | Declining | Stable | Unclear** (not emoji required). **Unclear** when evidence conflicts or is missing.

**Sources:** Every material claim should be traceable to a listed source with a clickable link when a URL exists.
