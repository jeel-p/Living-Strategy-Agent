# Examples

Use after the skill is loaded in OpenClaw or a compatible assistant with **search + fetch** enabled.

## Full report

```text
google_trends_report for https://example.com — full trend intelligence report with linked sources.
```

## Executive focus

```text
Using the google_trends_report skill: analyze https://ourcompany.io and give me the top 5 rising trends
that match our services, plus risks. Minimum searches as allowed by your tool budget.
```

## With market hint

```text
Trend report for https://brand.co — primary market UAE and KSA. Fetch the site, then run your searches.
```

## Sparse site (expect follow-up)

```text
Check trends for https://minimal-site.com — if the site is empty, ask me for our top services before
you finalize keywords.
```
