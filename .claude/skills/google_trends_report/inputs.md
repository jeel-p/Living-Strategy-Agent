# Inputs

## Required

| Input | Notes |
|-------|--------|
| **Company website URL** | HTTPS homepage; apex or www. Without a URL, ask for one or a clear brand + market. |

## Optional (ask if missing and site is sparse)

| Input | Notes |
|-------|--------|
| **Primary market / country** | Refines search language and regional trends. |
| **Top 2–3 services or products** | When the site has little text. |
| **Known competitors** | Improves competitor/alternative keyword searches (only use verified names). |
| **Report depth** | Default: full report; user may ask for executive summary only—still keep Sources. |

## Defaults

- **Report date:** Use **today’s date** (authoritative from the user or environment) for the metadata line and search query years.
- **Search rounds:** Target **12–18**; reduce only if the host limits tools, and disclose.
